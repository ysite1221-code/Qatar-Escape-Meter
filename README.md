# ①課題名
Qatar Escape Meter

## ②課題内容（どんな作品か）
- 2016年のQatar赴任時に、まじで嫌で嫌で毎日帰りたくてExcelで残り日数をカウントしていたものを、アプリ化
- 今回も完全に無用の長物であり、７：３で大喜利

## ⑤工夫した点・こだわった点
- LocalStrage保存の際のKeyが同一になり、上書きされてしまう問題：タイムスタンプをつける方式でKeyを挙動ごとにユニークにしたが、テキストとポイントがバラバラになり、ログでソートする時に一緒に表示されなかったので、テキスト、ポイント、日付を一緒にして保存することにしたらできた。
- 地味に外部からとってきた天気ウィジェットをセンタリングするのに超苦労。原因がわからなかったが、どうやら元々の属性でalign-leftが刺さっていた模様。結局、親要素を無理やり作ってそれをセンター寄せ、かつimportantをつける力技で上書きし、センター寄せ。ジャスティファイコンテントセンターの魔法が使えるようになりたい

## ⑥難しかった点・次回トライしたいこと（又は機能）
- やはり一番は視認性のところがまだまだ。使い勝手と見やすさをもっと突き詰めたい
- その他、やりたかったけど技量に対して時間足らずの部分ばかり。メーターは画像とか入れてカタール→日本で遊びたかった、過去ログを別ページに飛ばして個別削除をするようにしたかった、トップ５の表示を五件以下の場合は最小件数に応じて表示するとかやりたかった、など数えきれず。
- 配列がちゃんと使えなかったので次回配列を使いたい。
